THE GAME MACHINE

Return to the Retro

โครงงาน "The Game Machine" จัดทำขึ้นเพื่อเป็นการแสดงความรู้ภายหลังจากที่ได้ศึกษาวิชา computer programming มาตั้งแต่เริ่มภาคการศึกษาที่ 2 ปีการศึกษา 2563 และเพื่อเป็นการทดสอบความรู้ความเข้าใจ ของตัวผู้จัดทำเอง โครงงานชิ้นนี้เกี่ยวกับ ตัวเกมที่ถูกสร้างขึ้นด้วยภาษา C ทั้งหมดเพียงอย่างเดียว มีจำนวนทั้งสิ้น 2 เกมด้วยกัน วิธีในการจัดทำโครงงานนี้นั้น ทางผู้จัดทำ ได้ทำการ ค้นหาโค้ดต้นแบบ หรือ Source code มาทำการดัดแปลง และ แก้ไข รวมไปถึง โค้ดเกม ที่ทางผู้จัดทำ ได้เขียนขึ้นมาเอง โดยที่ไม่มีโค้ดต้นแบบมาเป็นส่วนประกอบ ร่วมด้วย ปัญหาที่เกิดขึ้น ณ ช่วงเวลาทำงาน ทางเราได้ใช้วิธีการ ค้นหาข้อมูลเพิ่มเติมผ่านทางอินเตอร์เน็ต ตัวเกมต้นแบบ ใช้เวลารวมทั้งสิ้น สามวัน โดยประมาณ และได้นำตัวเกมต้นแบบนี้ ให้ผู้เล่นได้ทดลองเพื่อหาข้อบกพร่องที่เกิดขึ้น เพื่อนำมาแก้ไขปรับปรุง จนในท้ายที่สุด ก็ได้ ตัวเกมตัวเต็ม ที่อยู่ในระดับที่น่าพึงพอใจ สิริเวลารวมทั้งสิ้นประมาณ หนึ่งสัปดาห์ โครงงานชิ้นนี้ได้บรรลุตามวัตถุประสงค์ที่ตั้งไว้ และ ยังสามารถนำไปต่อยอด โดยการเพิ่มเกมเข้าไป นอกเหนือจากที่มีอยู่อีกด้วย
